#Sidebar Navigation Menu

1) Toggle menu using a checkbox

2) Font Awesome icons for a modern lookThis project is a simple sidebar navigation menu built using HTML, CSS, and Font Awesome icons. The sidebar includes navigation links, a close 
   button, and social media icons.

##Features
1) Responsive sidebar navigation
2) Google Fonts for better typography

3) Soical media links

##Technologies Used

1) HTML5

2) CSS3

3)Font Awesome

4)Google Fonts

##Installation

###Clone the repository or download the files:

git clone https://github.com/your-username/sidebar-navigation.git

Open index.html in your browser.

Usage

1)Click on the hamburger icon to open the sidebar.

2)Click on the close button (X) to hide the sidebar.

3)Navigate through different sections using the provided links.
